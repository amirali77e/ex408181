# پروژه اشکال هندسی با PHP

این پروژه برای تمرین مدرسه ساخته شده است. هدف اصلی یادگیری استفاده از `const` و `abstract` در کلاس‌های PHP بود.

## توضیحات پروژه
- محاسبه مساحت مستطیل و دایره
- استفاده از کلاس انتزاعی و ثابت برای ساختار بهتر کد

## نحوه کار
- یک کلاس انتزاعی به نام `Shape` داریم که شامل یک ثابت برای واحد (سانتی‌متر مربع) و یک متد انتزاعی `area()` است.
- کلاس‌های `Rectangle` و `Circle` (در کد با نام `cricle`) این کلاس را گسترش داده و متد `area()` را برای هر شکل پیاده‌سازی می‌کنند.

## دلیل انجام پروژه
این پروژه توسط معلم برای یادگیری مفاهیم `const` و `abstract` در PHP داده شد. پروژه با موفقیت انجام شد و خروجی صحیح دارد.
